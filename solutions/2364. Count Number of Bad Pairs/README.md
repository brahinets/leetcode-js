# 2364. Count Number of Bad Pairs

## Medium

You are given a 0-indexed integer array `nums`. A pair of indices `(i, j)` is a bad pair if `i < j` and
`j - i != nums[j] - nums[i]`. Return the total number of bad pairs in `nums`.

### Constraints:

- `1 <= nums.length <= 10^5`
- `1 <= nums[i] <= 10^9`
