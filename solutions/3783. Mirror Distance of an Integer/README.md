# 3783. Mirror Distance of an Integer

## Easy

Given an integer `n`, return the **mirror distance** of `n`, defined as `abs(n - reverse(n))`.

`reverse(x)` denotes the integer formed by reversing the digits of `x`. Leading zeros are omitted after reversing (for example, `reverse(10) = 1`).

### Constraints:

- `1 <= n <= 10^9`
