# 15. 3Sum

## Medium

Given an integer array nums, return all the triplets `[nums[i], nums[j], nums[k]]` such that `i != j`, `i != k`,
and `j != k`, and `nums[i] + nums[j] + nums[k] == 0`. Notice that the solution set must not contain duplicate triplets.

### Constraints:

`3 <= nums.length <= 3000`
`-10^5 <= nums[i] <= 10^5`
