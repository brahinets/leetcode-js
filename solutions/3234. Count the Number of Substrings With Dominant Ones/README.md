# 3234. Count the Number of Substrings With Dominant Ones

## Medium

You are given a binary string `s`. Return the number of substrings with dominant ones. A string has dominant ones if the
number of ones in the string is greater than or equal to the square of the number of zeros in the string.

### Constraints:

- `1 <= s.length <= 4 * 10^4`
- `s` consists only of characters `0` and `1`
