# 3212. Count Submatrices With Equal Frequency of X and Y

## Medium

Given a 2D character matrix `grid` where each element is `'X'`, `'Y'`, or `'.'`, return the number of submatrices that:

1. Contain `grid[0][0]`
2. Have an equal frequency of `'X'` and `'Y'`
3. Contain at least one `'X'`

### Constraints:

- `1 <= grid.length, grid[0].length <= 1000`
- `grid[i][j]` is either `'X'`, `'Y'`, or `'.'`
