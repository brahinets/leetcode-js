# 3739. Count Subarrays With Majority Element II

## Hard

Given an integer array `nums` and an integer `target`, return the count of subarrays where `target` is the majority element.

The **majority element** of a subarray is the element that appears **strictly more than half** of the times in that subarray.

### Constraints:

- `1 <= nums.length <= 10^5`
- `1 <= nums[i] <= 10^9`
- `1 <= target <= 10^9`
