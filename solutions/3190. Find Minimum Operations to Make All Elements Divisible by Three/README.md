# 3190. Find Minimum Operations to Make All Elements Divisible by Three

## Easy

You are given an integer array `nums`. In one operation, you can add or subtract `1` from any element of `nums`. Return
the minimum number of operations to make all elements of `nums` divisible by `3`.

### Constraints:

- `1 <= nums.length <= 50`
- `1 <= nums[i] <= 50`
