# 1437. Check If All 1's Are at Least Length K Places Away

## Easy

Given an binary array `nums` and an integer `k`, return `true` if all `1`'s are at least `k` places away from each
other, otherwise return `false`.

### Constraints:

- `1 <= nums.length <= 10^5`
- `0 <= k <= nums.length`
- `nums[i]` is `0` or `1`
