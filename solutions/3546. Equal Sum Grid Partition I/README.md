# 3546. Equal Sum Grid Partition I

## Medium

Given an `m x n` matrix `grid` of positive integers, determine if you can make **either one horizontal or one vertical cut** such that each resulting section is non-empty and both sections have equal sums.

Return `true` if such a partition exists; otherwise return `false`.

### Constraints:

- `1 <= m == grid.length <= 10^5`
- `1 <= n == grid[i].length <= 10^5`
- `2 <= m * n <= 10^5`
- `1 <= grid[i][j] <= 10^5`
