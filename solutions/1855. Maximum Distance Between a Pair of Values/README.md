# 1855. Maximum Distance Between a Pair of Values

## Medium

Given two non-increasing integer arrays `nums1` and `nums2`, return the **maximum distance** `j - i` such that `i <= j` and `nums1[i] <= nums2[j]`, or `0` if no valid pair exists.

### Constraints:

- `1 <= nums1.length, nums2.length <= 10^5`
- `1 <= nums1[i], nums2[j] <= 10^5`
- Both `nums1` and `nums2` are non-increasing
