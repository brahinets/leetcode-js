# 3381. Maximum Subarray Sum With Length Divisible by K

## Medium

You are given an array of integers `nums` and an integer `k`. Return the maximum sum of a subarray of `nums`, such that
the size of the subarray is divisible by `k`.

### Constraints:

- `1 <= k <= nums.length <= 2 * 10^5`
- `-10^9 <= nums[i] <= 10^9`
