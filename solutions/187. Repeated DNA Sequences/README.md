# Repeated DNA Sequences

## Medium

The DNA sequence is composed of a series of nucleotides abbreviated as 'A', 'C', 'G', and 'T'.

For example: "ACGAATTCCG" is a DNA sequence.

When studying DNA, it is sometimes useful to identify repeated sequences within the DNA.

Given a string s that represents a DNA sequence, return all 10-letter-long sequences (substrings) that occur more than once in a DNA molecule. You may return the answer in any order.

**Constraints:**
- 1 <= s.length <= 10^5
- s[i] is either 'A', 'C', 'G', or 'T'.
- The answer should not contain duplicate sequences.
