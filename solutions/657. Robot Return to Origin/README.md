# 657. Robot Return to Origin

## Easy

A robot starts at position `(0, 0)` on a 2D plane. Given a string `moves` of characters `'U'`, `'D'`, `'L'`, `'R'`, return `true` if the robot returns to the origin after completing all moves, or `false` otherwise.

### Constraints:

- `1 <= moves.length <= 2 * 10^4`
- `moves` only contains the characters `'U'`, `'D'`, `'L'`, and `'R'`
